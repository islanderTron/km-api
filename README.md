# Swagger generated server

### Running the server
To run the server, run:

```
npm start
```

### The URI is:
```
http://localhost:8000
```

### The list of HTTP methods in API: 
```
GET /initalize
```

```
POST /node-clicked
```

```
POST /invalid-start-node
```

```
POST /valid-end-node
```

```
POST /invalid-end-node
```

```
POST /game-over
```
